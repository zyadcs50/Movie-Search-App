# Movie Search App

## Overview
The Movie Search App is a user-friendly application designed to help users find information about their favorite movies. It allows users to search for movies using various filters and provides detailed information about each movie, including cast, crew, synopsis, and ratings.

## Features
- **Search Functionality:** Search for movies by title, genre, or release year.
- **Detailed Information:** View comprehensive details about each movie, including plot summary, release date, cast, and ratings.
- **Responsive Design:** Optimized for both desktop and mobile views.
- **Bookmarking:** Save favorite movies for quick access later.

## Tech Stack
- **Frontend:** React.js for building user interfaces.
- **Backend:** Node.js and Express.js for server-side logic.
- **Database:** MongoDB for storing movie data.
- **APIs:** Integration with third-party movie databases (e.g., The Movie Database API).
- **Styling:** CSS and Bootstrap for responsive design.

## Setup Instructions
1. **Clone the repository:**
   ```bash
   git clone https://github.com/zyadcs50/Movie-Search-App.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd Movie-Search-App
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Configure Environment Variables:**
   Create a `.env` file in the root directory and add your API keys and database URI.
5. **Run the application:**
   ```bash
   npm start
   ```
   The app will be available at `http://localhost:3000`.

## Usage
Once the app is running, users can:
- Search for movies using the search bar.
- Click on a movie to view detailed information.
- Bookmark their favorite movies for easy access.
