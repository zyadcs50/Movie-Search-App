# 🎬 Movie Search App

A simple and responsive movie search application that allows users to search for movies and view their details using the **OMDb API**.

---

## 🚀 Features

- 🔍 Search for movies by title  
- 🎞 Display popular movies on first load  
- 📄 View detailed information about each movie  
- 💾 Save last search results using `localStorage`  
- 🧹 Clear search history  
- ⬆ Scroll to top button  
- 📱 Responsive design  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Flexbox & Grid)  
- JavaScript (Vanilla JS)  
- OMDb API
